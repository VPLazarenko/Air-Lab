ТЕСТОВЫЙ ФАЙЛ ДЛЯ БАЗЫ ЗНАНИЙ

Этот документ содержит специфическую информацию о работе с Drizzle ORM и PostgreSQL:

1. НАСТРОЙКА DRIZZLE ORM:
- Используйте drizzle-kit для миграций: npm run db:push
- Создайте схемы в shared/schema.ts
- Настройте подключение через @neondatabase/serverless

2. БЕЗОПАСНОСТЬ POSTGRESQL:
- ВСЕГДА используйте подготовленные запросы (prepared statements)
- Никогда не вставляйте пользовательские данные напрямую в SQL
- Используйте параметризированные запросы через Drizzle ORM
- Настройте правильные роли и права доступа

3. ИНДЕКСЫ В POSTGRESQL:
- Создавайте индексы для часто используемых WHERE и JOIN условий
- Используйте EXPLAIN ANALYZE для анализа производительности
- Составные индексы для многоколоночных запросов
- Регулярно обновляйте статистику через ANALYZE

4. СПЕЦИФИЧЕСКИЕ КОМАНДЫ DRIZZLE:
- eq() для точного сравнения
- like() для поиска по шаблону  
- and(), or() для сложных условий
- select().from().where() для запросов
